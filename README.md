# The Latent Space

This is my personal blog where I write about the technical things I am learning and interested in, specifically deep learning, neural networks, large language models (LLMs), and neuroscience. 

I built this site to have a clean, focused place to share deep dives and math breakdowns without the noise of normal social media.

## 🛠️ The Tech Stack

I chose these tools to keep the site fast, clean, and highly readable:
* **React & TypeScript**: For building a fast interface with reliable, bug-free code.
* **Tailwind CSS**: For a clean, dark-themed developer notebook look.
* **KaTeX**: A lightweight library that reads math equations (LaTeX) in my text files and converts them into clean formulas.

##  Current Posts Inside

1. **The Token Routing Crisis**: A post looking at the math behind Mixture-of-Experts (MoE) models, how tokens move across different GPUs, and why hardware limits are shifting from processing power to network speeds.
2. **The Synaptic Divergence**: A short comparison looking at how artificial neural networks compare to biological brains in terms of power efficiency and learning.

##  How to Run it Locally

If you want to clone this project and run it on your machine, follow these steps:

1. Clone the repository:
```bash
git clone [https://github.com/YOUR_GITHUB_USERNAME/the-latent-space.git](https://github.com/YOUR_GITHUB_USERNAME/the-latent-space.git)
cd the-latent-space

```

2. Install the project dependencies:

```bash
npm install

```

3. Start the local development server:

```bash
npm run dev

```

Open your browser and head to `http://localhost:5173` to see it running!
